# 赚了么 - 微信记账小程序

一款帮助用户记录彩票、刮刮乐、麻将收支的微信小程序，支持全用户排行榜功能。

## 功能特性

- 📊 **三种记账类型**：彩票、刮刮乐、麻将
- 📝 **记录管理**：支持新增、编辑、删除记录
- 📈 **个人统计**：年度总盈亏、分类盈亏、最近记录
- 🏆 **全用户排行榜**：总排行、彩票排行、刮刮乐排行、麻将排行
- 👤 **个人中心**：自定义昵称和头像
- ☁️ **微信云开发**：无需服务器，免费部署

## 技术栈

- 微信小程序原生开发
- 微信云开发（云函数 + 云数据库 MongoDB）
- Node.js 云函数

## 部署步骤

### 1. 创建云开发环境
- 登录微信开发者工具
- 创建小程序项目
- 开通云开发，记录环境ID

### 2. 配置环境ID
修改 `app.js`：
```javascript
wx.cloud.init({
  env: '你的环境ID',
  traceUser: true
});
```

### 3. 部署云函数
在微信开发者工具中：
1. 展开 `cloud/functions/`
2. 右键每个云函数文件夹
3. 选择 **"创建并部署：云端安装依赖"**
4. 依次部署6个云函数：
   - `user`
   - `lottery`
   - `scratch`
   - `mahjong`
   - `stats`
   - `rank`

### 4. 创建数据库集合
在云开发控制台 → 数据库，创建以下集合：
- `users` - 用户信息（_openid, nickname, avatarUrl, createTime, updateTime）
- `lottery` - 彩票记录（_openid, cost, winAmount, createTime, updateTime）
- `scratch` - 刮刮乐记录（_openid, cost, winAmount, createTime, updateTime）
- `mahjong` - 麻将记录（_openid, amount, createTime, updateTime）

**权限设置**：所有集合选择 **"仅创建者可读写"**

### 5. 配置图片资源
将 `images/logo.png` 上传到云存储，获取 fileID 并更新到项目配置中。

### 6. 运行测试
点击预览，测试各项功能。

## 核心功能说明

### 记账流程
1. 首页点击快速记账按钮（彩票/刮刮乐/麻将）
2. 输入花费金额和中奖金额（麻将选择赢/输）
3. 实时预览盈亏金额
4. 保存后自动更新统计

### 记录编辑
- 点击首页最近记录，弹出操作菜单
- 支持编辑和删除
- 编辑时自动带出原有数据

### 排行榜规则
- **总排行**：彩票盈亏 + 刮刮乐盈亏 + 麻将盈亏
- **分类排行**：单独计算各类型的盈亏
- 按盈亏金额从高到低排序
- 显示用户头像和昵称

### 用户系统
- 首次记账自动创建用户记录
- 默认昵称为"赚了么用户"
- 可在"我的"页面修改昵称和头像
